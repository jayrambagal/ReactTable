## ReactTable
### Building a Table in React using react-table and  react query for API queries in which user can:
* Create User 
* List users with
    *  sorting 
    *  pagination
* Delete user.
* Edit user - Can edit name and role
* Ability to download CSV
* Confirmation modal on deleting user
### Used Technology
  1. React
  2. TypeScript
  3. React table
  4. React Query
  5. Nodejs
  6. Expressjs
  7. MongoDB
